# swiftBookRecommend
起点小说推荐app


关于数据来源

使用 https://github.com/bravekingzhang/moveSpider 爬虫程序爬取数据，在之前的基础上做了稍微改造。

因为数据不是我们自己创造的，因此，仅供大家学习使用，不得用于任何商业行为。

所使用到的框架：

网络数据加载框架：Alamofire (3.2.0)
图片加载框架：SDWebImage (3.7.5)
json解析框架：SwiftyJSON (2.3.2)
toast：HRToast
